Main Description
Cada detalle del flujo de trabajo representa una habilidad clave que debe aplicarse para realizar una gestión de requisitos eficaz. Analizar el problema y comprender las necesidades de las partes interesadas se centran durante la fase inicial de un proyecto, mientras que el énfasis está en definir el sistema y perfeccionar la definición del sistema durante la fase de elaboración . La gestión del alcance del sistema y la gestión de los requisitos cambiantes se realizan de forma continua durante todo el proyecto.
Los detalles del flujo de trabajo se muestran en un orden secuencial lógico. Como se indica en el texto anterior, se aplican continuamente en orden variado según sea necesario a lo largo del proyecto. Aquí se muestran en la secuencia que probablemente aplicaría a la primera iteración de un nuevo proyecto.
