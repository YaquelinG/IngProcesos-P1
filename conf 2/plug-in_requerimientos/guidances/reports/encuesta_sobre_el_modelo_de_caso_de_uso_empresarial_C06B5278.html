Report: encuesta sobre el modelo de caso de uso empresarial
Este informe describe el modelo de caso de uso empresarial. Ofrece una descripción completa de los resultados del modelado de casos de uso e incluye descripciones breves para cada actor empresarial y caso de uso empresarial.

Main Description

1. Introducción
Una breve descripción del propósito del modelo de caso de uso.

2. Descripción de la encuesta
Una lista resumida de actores y casos de uso tal como se organizan a continuación. Esta sección también debe describir datos básicos sobre el negocio, como la idea de negocio y los mercados en los que opera.

3. Jerarquía del modelo de casos de uso empresarial
Esta sección presenta los paquetes de casos de uso de forma jerárquica, explica las dependencias entre ellos y muestra el contenido de cada paquete de forma recursiva. Si el modelo tiene varios niveles de paquetes, los del nivel superior se presentan primero. Los paquetes dentro de estos se presentan a continuación, y así sucesivamente, hasta llegar a los paquetes en la parte inferior de la jerarquía. Para cada paquete incluye:

Su nombre.
Una breve descripción que explica la función y el papel del paquete en el sistema. La descripción debe ser comprensible para cualquier desarrollador que quiera utilizar el paquete.
Una lista de los casos de uso que posee el paquete, incluido el nombre y una breve descripción de cada caso de uso.
Una lista de actores propiedad del paquete, incluido el nombre y una breve descripción de cada actor.
Una lista de relaciones propiedad del paquete, incluido el nombre y una breve descripción de cada relación.
Una lista de los paquetes que pertenecen directamente al paquete, con cada paquete presentado de la misma manera jerárquica que antes.
4. Diagramas del modelo de casos de uso empresarial
Aquí se incluyen diagramas, principalmente diagramas de casos de uso, de todo el modelo de casos de uso.
