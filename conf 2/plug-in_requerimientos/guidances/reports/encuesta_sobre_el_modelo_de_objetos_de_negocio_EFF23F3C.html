Main Description

1. Introducción
Una introducción al modelo de objetos de negocio.

2. Unidades organizativas
Esta sección presenta las unidades organizativas jerárquicamente, explica las dependencias entre ellas y muestra el contenido de cada unidad organizativa de forma recursiva. Si el modelo tiene varios niveles de unidades organizativas, las del nivel superior se presentan primero. Las unidades organizativas dentro de estos se presentan a continuación, y así sucesivamente, hasta llegar a las unidades organizativas en la parte inferior de la jerarquía. Para cada unidad organizativa incluya:

Su nombre.
Una breve descripción.
Una lista de los trabajadores comerciales y las entidades comerciales propiedad de la unidad organizativa, incluido el nombre y una breve descripción.
Una lista de las relaciones propiedad de la unidad organizativa, incluido el nombre y una breve descripción de cada relación.
Una lista de las unidades organizativas de propiedad directa de la unidad organizativa, cada una presentada de la misma manera jerárquica que se muestra arriba.
3. Diagramas del modelo de objetos de negocio
Aquí se incluyen los diagramas, principalmente diagramas de clases, de todo el modelo de objetos de negocio. Las realizaciones de casos de uso empresarial y sus diagramas se informan por separado en Informe: Realización de casos de uso empresarial.
