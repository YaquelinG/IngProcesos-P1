Tool Mentor: Creación de una encuesta de modelo de caso de uso utilizando Rational SoDA
Este mentor de herramientas describe cómo utilizar Rational SoDA para crear una encuesta de modelo de caso de uso. SoDA automatiza la generación del informe para que se cree de forma rápida y precisa. Puede generar una encuesta de modelo de caso de uso con la versión Microsoft Word o Adobe FrameMaker de SoDA. Esto sólo funciona si el modelo de Rational Rose sigue la estructura y la convención de nomenclatura del modelo de casos de uso.

Main Description

Esta herramienta mentora es aplicable cuando se ejecuta Windows 98, Windows 2000, NT 4.0, Windows XP, Solaris o HP-UX.

Para crear un informe de encuesta de modelo de caso de uso utilizando SoDA, utilice el procedimiento para su versión del producto:

Utilice Rational SoDA/Word para generar una encuesta de modelo de caso de uso
Utilice Rational SoDA/FrameMaker para generar una encuesta de modelo de caso de uso
Utilice Rational SoDA/Word para generar una encuesta de modelo de caso de uso
Desde cualquier lugar de Rational Rose, haga clic en Informe > Informe SoDA .
Cuando aparezca la lista de informes disponibles en SoDA, seleccione Encuesta del modelo de caso de uso de proceso unificado de Rational .
Haga clic en Aceptar para generar el informe.
Utilice Rational SoDA/FrameMaker para generar una encuesta de modelo de caso de uso Ir al principio de la página
Desde la barra de botones de FrameMaker, haga clic en Nuevo . Haga doble clic en SoDA , luego haga doble clic en RoseDomain y elija la plantilla RUPUseCaseModelSurvey.fm .
Edite el conector e ingrese el nombre del archivo del modelo.
Haga clic en Archivo > Guardar como para guardar la plantilla en un directorio personal o de proyecto.
Haga clic en SoDA > Generar documento .
Revisar el documento generado.
La próxima vez que desee generar este mismo documento, simplemente abra el documento y haga clic en SoDA > Generar documento .
