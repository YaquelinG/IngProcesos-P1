Artifact: Lista de riesgos
Una lista ordenada de riesgos conocidos y abiertos para el proyecto, ordenados en orden decreciente de importancia y asociados con acciones específicas de mitigación o contingencia.
Purpose
La Lista de Riesgos está diseñada para capturar los riesgos percibidos para el éxito del proyecto. Identifica, en orden decreciente de prioridad, los eventos que podrían conducir a un resultado negativo significativo. Sirve como punto focal para las actividades del proyecto y es la base alrededor de la cual se organizan las iteraciones.

Description
Main Description

La Lista de Riesgos se mantiene durante todo el proyecto. Se crea temprano en la fase inicial y se actualiza continuamente a medida que se descubren nuevos riesgos y los riesgos existentes se mitigan o eliminan. Como mínimo, se revisa al final de cada iteración, a medida que se evalúa la iteración.

Responsabilidad
El Gerente de Proyecto es responsable de mantener la Lista de Riesgos y mantenerla actualizada.

Sastrería
La Lista de Riesgos debe capturar los riesgos críticos y graves; si encuentra que esta lista se extiende más allá de 20, considere cuidadosamente si son riesgos realmente graves. El seguimiento de más de 20 riesgos es una tarea onerosa.
