Artifact: Modelo de objetos de negocio
El modelo de objetos de negocio es un modelo de objetos que describe la realización de casos de uso de negocios.
Purpose

Las siguientes personas utilizan el modelo de objetos de negocio:

El analista de procesos de negocio es responsable de la integridad del modelo.
El diseñador de negocios documenta detalles sobre las responsabilidades que tienen las personas y las "cosas" producidas o utilizadas por la empresa en este modelo.
El analista de sistemas utiliza este modelo como entrada para identificar los actores del sistema y los casos de uso del sistema.
El diseñador utiliza este modelo como entrada para identificar clases en los modelos de análisis y diseño.
