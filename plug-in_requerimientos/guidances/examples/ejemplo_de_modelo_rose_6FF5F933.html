Main Description

El modelo Rose de ejemplo (publicado por Rose Web Publisher) para Collegiate Sports Paging System debe instalarse por separado.

Siga las instrucciones a continuación para instalar los archivos de ejemplo en la carpeta de ejemplos de Rational Unified Process (RUP):

Para descomprimir el archivo csports_addition.zip , necesitará instalar WinZip en su PC como utilidad de extracción predeterminada. Si aún no tiene una copia de WinZip, vaya a http://www.winzip.com/ para descargar una copia.

Haga clic en el enlace csports_addition.zip y extraiga todos los archivos al directorio donde está instalado su RUP. Por ejemplo, d:\Program Files\Rational\RationalUnifiedProcess.

Responda "Sí" cuando se le solicite reemplazar el archivo ..../root.html.

Cuando se complete la extracción, actualice esta página.
