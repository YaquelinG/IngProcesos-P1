Example: Sistema de registro de cusos
El Proyecto del Sistema de Registro de Cursos es un proyecto ficticio para desarrollar un sistema de registro de cursos en línea para Wylie College. El proyecto pretende reemplazar el sistema de registro de cursos existente con un sistema en línea de última generación que permita el acceso de estudiantes y profesores a través de clientes de PC.

El ejemplo del Proyecto del sistema de registro de cursos es parte del ejemplo web de la organización Wylie College , que muestra cómo una organización puede crear dos sitios web complementarios, uno para la implementación del proceso unificado de Rational por parte de la organización y otro para un proyecto específico que utiliza esa implementación.
