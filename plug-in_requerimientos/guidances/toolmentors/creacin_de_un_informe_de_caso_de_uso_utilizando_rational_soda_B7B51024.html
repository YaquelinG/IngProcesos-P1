Tool Mentor: Creación de un informe de caso de uso utilizando Rational SoDA
Este mentor de herramientas describe cómo utilizar Rational SoDA para crear un informe de caso de uso. SoDA automatiza la generación del informe para que se cree de forma rápida y precisa. Puede generar un informe de caso de uso con la versión Microsoft Word o Adobe FrameMaker de SoDA.

Main Description

Esta herramienta mentora es aplicable cuando se ejecuta Windows 98, Windows 2000, NT 4.0 o Windows XP, Solaris o HP-UX.

Para crear un informe de caso de uso utilizando SoDA, utilice el procedimiento para su versión del producto:

Utilice Rational SoDA/Word para generar un informe de caso de uso
Utilice Rational SoDA/FrameMaker para generar un informe de caso de uso
Utilice Rational SoDA/Word para generar un informe de caso de uso
En un diagrama de casos de uso en Rational Rose, seleccione el caso de uso que desea documentar.
Haga clic en Informe > Informe SoDA .
Cuando aparezca la lista de informes disponibles en SoDA, seleccione Informe de caso de uso de Rational Unified Process .
Si este informe no está en la lista, cancele y asegúrese de tener un caso de uso seleccionado en el diagrama de casos de uso.
Haga clic en Aceptar .
Cuando se le solicite la especificación de caso de uso, ingrese o busque el archivo de Word que se incluirá en este informe.
Haga clic en Aceptar para generar el informe.
2. Utilice Rational SoDA/FrameMaker para generar un informe de caso de uso
Desde la barra de botones de FrameMaker, haga clic en Nuevo . Haga doble clic en SoDA , luego haga doble clic en RoseDomain y elija la plantilla RUPUseCaseReport.fm .
Edite el primer Conector e ingrese el nombre del modelo, paquete y caso de uso.
Haga clic en Archivo > Guardar como para guardar la plantilla en un directorio personal o de proyecto. Es posible que desee cambiar el nombre de la plantilla para reflejar el nombre del caso de uso; por ejemplo, MantenerLoansReport.fm.
Haga clic en SoDA > Generar documento .
Cuando se le solicite la especificación de caso de uso, ingrese o busque el archivo FrameMaker que se incluirá en este informe.
Haga clic en Aceptar para generar el informe.
Revisar el documento generado.
La próxima vez que desee generar este mismo documento, ábralo y haga clic en SoDA > Generar documento .
