Artifact: Directrices de interfaz de usuario
Directrices específicas del proyecto sobre cómo crear la interfaz de usuario.
Purpose

Las siguientes personas utilizan las Directrices de la interfaz de usuario:

Diseñadores de interfaces de usuario cuando modelan y crean prototipos de la interfaz de usuario.
Diseñadores e Implementadores cuando implementan la interfaz de usuario.

Description
Main Description

Las Pautas de la interfaz de usuario deben brindar reglas y recomendaciones específicas del proyecto para construir la interfaz de usuario, si es necesario. Esto es importante si una gran parte del sistema depende de la interfaz de usuario y si se enfatiza la usabilidad.

Estas pautas también pueden hacer referencia a publicaciones externas, como The Windows Interface Pautas para el diseño de software, de Microsoft® Corporation.

Momento
Las pautas de la interfaz de usuario se crean al principio de la fase inicial.

Responsabilidad
Un diseñador de interfaz de usuario es responsable de la integridad de estas pautas, asegurando que satisfagan las necesidades del proyecto.

Sastrería
Las pautas de la interfaz de usuario deben adaptarse para satisfacer las necesidades del proyecto.
